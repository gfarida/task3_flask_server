Выбор алгоритма

Привет! Этот сайт предназначен для быстрого взаимодействия с моделями машинного обучения. Вы сможете создать модель, выбрать для нее нужные гиперпараметры, загрузить свой датасет и получить предсказание для целевой переменной! Также можно узнать значение RMSE на каждой итерации обучения и др.
Для начала выберите алгоритм, доступны два - Random Forest (случайный лес), Gradient Boosting (градиентный бустинг над деревьями).
